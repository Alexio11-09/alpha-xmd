- 🔌 Advanced plugin system (drop-in commands)

- 👑 Owner & admin control system

- 📂 Organized command categories (AI, Downloader, Tools, etc.)

- 📢 Channel-style messaging system
- 🖼️ Global thumbnail branding
- 👁️ Auto-read, typing & reaction system
- 🔥 Fast, lightweight & optimized performance

⚙️ Configuration

All core settings are managed in:

settings/config.js

You can customize:

- 👑 Owner number
- 🤖 Bot name
- 🖼️ Thumbnail
- ⚡ Prefix
- 📢 Channel integration
- 🌍 API settings
- 🔥 Feature toggles (auto read, typing, etc.)

---

📦 Plugin System

Adding commands is simple:

plugins/your-command.js

Each plugin:

- Has its own command
- Supports categories
- Loads automatically
- Supports permissions (owner/admin/group)

«⚡ No restart needed — plugins reload instantly.»
