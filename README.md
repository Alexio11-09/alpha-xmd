## 🔑 How to Get a Session ID

You need a **Session ID** so the bot can connect to WhatsApp without scanning a QR code every time.

<div align="center">
  <a href="https://alpha-xmd.vercel.app">
    <img src="https://img.shields.io/badge/📱_PAIRING_WEBSITE-Get_Your_Code_Here-3b82f6?style=for-the-badge" alt="Pairing Website"/>
  </a>
</div>

### 📖 Steps:
1. **Visit** [alpha-xmd.vercel.app](https://alpha-xmd.vercel.app)
2. **Enter** your WhatsApp number (with country code)
3. **Get** your 8-digit pairing code
4. **Open** WhatsApp → Settings → Linked Devices → Link a Device
5. **Enter** the code → **CONNECTED!**

## ✨ Features

### 📥 Media & Downloads
| Command | Function |
|---------|----------|
| `.play` | Search & play audio from YouTube |
| `.tiktok` | Download TikTok videos (no watermark) |
| `.ig` | Download Instagram reels/photos |
| `.fb` | Download Facebook videos |
| `.mediafire` | Download MediaFire files |
| `.twitter` | Download Twitter/X videos |
| `.apk` | Download Android APK files |
| `.movie` | Search movie info & posters |
| `.wallpaper` | Download HD wallpapers |
| `.gitclone` | Download GitHub repositories |

### 👥 Group Management
| Command | Function |
|---------|----------|
| `.tagall` | Tag all group members |
| `.kick` | Remove member from group |
| `.add` | Add member to group |
| `.promote` | Make member admin |
| `.demote` | Remove admin status |
| `.mute` / `.unmute` | Mute/unmute group |
| `.welcome` | Toggle welcome messages |
| `.goodbye` | Toggle goodbye messages |
| `.antilink` | Anti-link protection (delete/warn/kick) |
| `.groupinfo` | Show group details |
| `.grouplink` | Get group invite link |
| `.listadmin` | List all group admins |
| `.tagadmin` | Tag all admins only |

### 🛡️ Protection Suite
| Command | Function |
|---------|----------|
| `.antilink on/off` | Enable/disable link blocking |
| `.antilink mode` | Set action (delete/warn/kick) |
| `.antidelete` | Detect deleted messages |
| `.antiedit` | Detect edited messages |
| `.block` / `.unblock` | Block/unblock users |
| `.banuser` / `.unbanuser` | Ban users from bot |

### 🎮 Fun & Games
| Command | Function |
|---------|----------|
| `.tictactoe` | Play Tic-Tac-Toe |
| `.guess` | Number guessing game |
| `.quiz` | Trivia quiz |
| `.riddle` | Solve riddles |
| `.truth` / `.dare` | Truth or Dare |
| `.joke` | Random jokes |
| `.quote` | Inspirational quotes |

### 👑 Owner Tools
| Command | Function |
|---------|----------|
| `.restart` | Restart bot |
| `.shutdown` | Stop bot |
| `.eval` | Execute JavaScript |
| `.bc` | Broadcast message |
| `.bcgc` | Broadcast to groups |
| `.join` | Join group via link |
| `.leave` | Leave group |
| `.pm` | Send private message |
| `.addowner` | Add temporary owner |

### ⚙️ Settings
| Command | Function |
|---------|----------|
| `.autoread` | Auto-read messages |
| `.autotyping` | Show typing indicator |
| `.autorecording` | Show recording indicator |
| `.autoreact` | Auto-react to messages |
| `.autoviewstatus` | Auto-view statuses |
| `.autoreactstatus` | Auto-react to statuses |
| `.setprefix` | Change command prefix |
